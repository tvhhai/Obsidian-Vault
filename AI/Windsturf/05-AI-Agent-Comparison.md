## 4. Ma Trận Quyết Định

### 4.1 Chọn Agent Theo Nhu Cầu

```
Bạn cần gì?
│
├── Daily coding, quick iterations
│   ├── VS Code user → Copilot or Cascade
│   ├── JetBrains user → JetBrains AI or Copilot
│   └── Terminal-first → Claude Code
│
├── Complex refactoring
│   ├── Git workflow → Aider
│   └── Planning-heavy → Claude Code or Cascade
│
├── Greenfield project
│   ├── From scratch → Replit Agent or Lovable
│   └── With requirements → Devin (if available)
│
├── Enterprise/Team
│   ├── AWS shop → Amazon Q
│   ├── Google/GCP shop → Gemini Code Assist / Firebase Studio
│   ├── GitHub shop → Copilot Enterprise
│   └── Self-hosted → Tabnine Enterprise
│
└── Open source / Free
    └── Aider (best free option)
```

### 4.2 Trade-offs Matrix

| Nếu bạn muốn... | Thì dùng... | Nhưng lưu ý... |
|-----------------|-------------|----------------|
| Native IDE feel | Cascade, Cursor, Copilot | Vendor lock-in |
| Maximum flexibility | Claude Code, Aider | CLI learning curve |
| Full autonomy | Devin, Replit Agent | Less control |
| Free + Open source | Aider | Setup required |
| Enterprise compliance | Amazon Q, Tabnine | Cost higher |
| Best completions | Copilot | No agent features |
